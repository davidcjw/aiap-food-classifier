Predicting Food Classes

About the model

This model is trained using the ResNet50 architecture using ImageNet weights, fine-tuned on the last 3 layers and adding a Dense layer containing 512 nodes, followed by an output layer.

We used the Adam optimiser with a learning rate of 0.001 and trained for 20 epochs with an EarlyStopping patience of 3. The validation accuracy we achieved at the end of training was 70%.

The model is able to handle both .jpeg and .png files.
